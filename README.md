# 1306 Cleaner Pro ALL stock

PWA สำหรับทำความสะอาดไฟล์สต็อก 1306 ทั้งคลังใหญ่และคลังย่อย ใช้งานผ่าน GitHub Pages และติดตั้งเป็นแอปบนคอมพิวเตอร์หรือโทรศัพท์ได้

## การใช้งาน

1. เลือกโหมด `คลังใหญ่` หรือ `คลังย่อย`
2. เลือกไฟล์ `.xls`, `.xlsx` หรือ `.csv`
3. กดประมวลผลและตรวจจำนวนรายการ
4. เลือกดาวน์โหลด Excel หรือกด `ส่งขึ้น Google Sheets`
5. การส่งขึ้นชีทจะแทนที่เฉพาะคอลัมน์ A:I และรักษาสูตรตั้งแต่ J2 เป็นต้นไป

การส่งขึ้น Google Sheets ต้องเปิดด้วยเบราว์เซอร์ที่ล็อกอินบัญชี Google ซึ่งมีสิทธิ์ใช้ Web App

## PWA

- เปิดเว็บด้วย Chrome หรือ Edge
- เลือก `ติดตั้งแอป` จากแถบที่อยู่หรือเมนูของเบราว์เซอร์
- ตัวแอปเก็บส่วนประกอบหลักไว้ในแคชเพื่อช่วยให้เปิดใช้งานได้เร็วขึ้น

ข้อมูลไฟล์ Excel ถูกประมวลผลในเบราว์เซอร์ของผู้ใช้ และจะส่งเฉพาะผลลัพธ์ Table เมื่อผู้ใช้กดยืนยันส่งขึ้น Google Sheets
